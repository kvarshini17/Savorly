Savorly – Java Recipe Manager

Overview:
Savorly is a desktop-based recipe manager built using Java Swing and file handling. It allows users to add, view, categorize, and favorite recipes. It also includes a BMI checker for health-conscious users. The app uses simple .txt files for data storage and supports image previews for recipes.

How to Run:
1. Extract all files from Savorly.zip into a single folder.
2. Open all .java files in your IDE (e.g., VS Code).
3. Compile and run RecipeManagerUI.java.
4. Ensure Recipe.txt, Favorite.txt, and Category.txt are in the same folder.
5. Place any required .png images in the same folder as the code.

Files Included:
- RecipeManagerUI.java – Main window and navigation
- ViewPage.java – Displays all recipes
- FavoritePage.java – Shows favorite recipes
- AddRecipePage.java – Form to add new recipes
- BMICheckerPage.java – BMI calculator
- CategoryWindow.java – Category-based filtering
- RecipeDetails.java – Detailed view of a selected recipe
- Recipe.txt – Stores all recipes
- Favorite.txt – Stores favorite recipe names
- Category.txt – Stores category mappings
- (Optional) .png images used in the GUI
